# VoltLine landing

Демо-лендінг для портфоліо: сервіс підбору та монтажу резервного живлення для українського ринку.

## Стек

- HTML
- CSS
- JavaScript
- GitHub Pages-ready

## Функції

- адаптивний перший екран з локальним hero-зображенням;
- пакети для квартири, будинку та бізнесу;
- інтерактивний калькулятор підбору системи;
- мобільне меню;
- форма заявки з простою валідацією.

## Запуск

Відкрийте `index.html` у браузері або опублікуйте папку через GitHub Pages.
